## Abstract

With the rapid development of artificial intelligence technology, Large Language Models (LLMs) have been increasingly applied in vertical domains. This paper designs and implements an intelligent nutrition advisor system based on LLM and Retrieval-Augmented Generation (RAG) technology, aiming to provide users with personalized nutrition consultation, recipe recommendation, and health management services. The system adopts a front-end and back-end separated architecture, with the front-end built using Vue 3 and Vant UI framework for mobile applications, and the back-end based on FastAPI framework providing RESTful API services. Core functionalities include: AI-powered nutrition consultation, RAG-based personalized recipe recommendation, multi-modal food ingredient recognition, seasonal recipe recommendation, health profile management, and a knowledge base management system with document parsing. The system innovatively combines traditional relational databases with vector databases to achieve unified management and integrated retrieval of structured business data and unstructured knowledge data. The document parsing module supports automatic chunking and vectorized storage of PDF, Word, and other formats, with asynchronous background task mechanism for non-blocking processing of long-running operations. Test results show that the system can effectively meet users' diverse needs in daily nutrition and health management, demonstrating good practical value and promotion prospects.

**Keywords:** Large Language Model; Retrieval-Augmented Generation; Nutrition Advisor; Vector Database; FastAPI; Vue 3

## 第 1 章 绪论

### 1.1 研究背景与意义

#### 1.1.1 研究背景

随着经济社会的发展和人民生活水平的提高，公众对健康的关注度日益提升，科学的营养管理已成为全民健康的重要组成部分。《中国居民膳食指南（2022）》指出，合理的膳食结构是预防慢性疾病、提升国民健康素养的关键手段[1]。然而，传统的营养咨询服务存在专业门槛高、服务成本高、覆盖面有限等问题，普通大众难以获得及时、个性化的营养指导。与此同时，随着智能手机和移动互联网的普及，用户越来越倾向于通过数字化工具获取健康管理服务，这为智能营养顾问系统的开发提供了广阔的应用场景。

近年来，人工智能领域取得了突破性进展，特别是大语言模型（Large Language Model, LLM）的出现，为自然语言理解和生成任务带来了革命性的变化。2017年，Vaswani等提出了基于自注意力机制的Transformer架构，为后续大语言模型的发展奠定了技术基础[2]。2020年，Brown等发布的GPT-3模型展示了大规模语言模型在 Few-Shot 学习方面的强大能力[3]。2022年底，ChatGPT的发布标志着大语言模型正式进入大众视野，其在对话理解、知识问答、文本生成等方面展现出的卓越能力，为构建智能化营养咨询服务提供了新的技术路径。

然而，大语言模型在垂直领域应用时仍面临诸多挑战。一方面，通用大语言模型缺乏专业领域的深度知识，在回答专业营养问题时可能出现事实性错误或"幻觉"现象；另一方面，模型的知识更新存在滞后性，难以实时获取最新的营养学研究进展和膳食指南。检索增强生成（Retrieval-Augmented Generation, RAG）技术的出现有效缓解了上述问题。RAG通过将外部知识检索与大语言模型的生成能力相结合，使模型能够基于权威、可验证的知识源生成回答，显著提升了回答的准确性和可信度[4]。Lewis等在其开创性工作中证明，RAG框架在知识密集型NLP任务中能够有效减少模型幻觉，提高生成内容的事实准确性[4]。

在营养健康领域，RAG技术的应用前景尤为广阔。研究表明，将大语言模型与可靠的营养知识库相结合，可以生成科学、个性化且符合语境的膳食建议[5]。例如，DietAI24框架通过结合多模态大语言模型与RAG技术，将视觉识别结果与权威营养数据库关联，实现了准确的营养估算[6]。ChatDiet系统则利用LLM增强的框架，整合个人健康档案与群体营养知识，提供可解释的个性化食物推荐[7]。这些研究表明，基于LLM与RAG技术的智能营养顾问系统具有重要的研究价值和应用潜力。

### 1.2 国内外研究现状

#### 1.2.1 大语言模型在健康领域的应用研究

大语言模型在医疗健康领域的应用已成为当前研究的热点之一。在医疗问答方面，Singhal等开发了Med-PaLM，这是一个专门面向医学领域的大语言模型，在美国医师执照考试（USMLE）中取得了接近专家水平的成绩[8]。随后，Med-PaLM 2进一步优化，在多项医学问答基准测试中表现优异[9]。在国内，复旦大学团队开发的DISC-MedLLM专注于医疗对话场景，通过构建高质量医疗指令数据集提升了模型在中文医疗咨询中的表现[10]。

在营养健康领域，大语言模型的应用研究也取得了积极进展。Zhang等提出了NutriBench数据集，用于评估大语言模型从膳食描述中估算营养成分的能力，实验结果表明GPT-4o配合思维链提示在营养估算任务中达到了66.82%的准确率，展现了LLM在营养分析领域的潜力[11]。另一项研究利用GPT-4o、Llama-3和Mixtral等模型进行复杂食物成分分解，显著提升了膳食规划的精确性[12]。然而，现有研究也指出，大语言模型在营养领域应用时存在推理能力有限、事实准确性不足、领域特异性欠缺等问题，且存在偏见和幻觉风险，需要严格的验证和人工监督[13]。

#### 1.2.2 检索增强生成技术的研究现状

检索增强生成（RAG）技术自2020年Lewis等提出以来，经历了快速的发展和演进。早期的RAG系统主要采用"检索-阅读"（Retrieve-Then-Read）的流水线架构，先从外部知识库中检索相关文档片段，再将其作为上下文输入生成模型以生成最终回答[4]。随着研究的深入，RAG技术逐渐发展出多种变体，包括迭代式RAG（Iterative RAG）、自适应RAG（Adaptive RAG）以及多跳推理RAG（Multi-hop RAG）等，以应对更复杂的知识密集型任务[14]。

在医疗健康领域，RAG技术的应用受到了广泛关注。Xiong等提出了MIRAGE基准测试，系统评估了RAG系统在医学问答任务中的表现，涵盖了来自五个医学问答数据集的7663个问题，实验涉及41种RAG配置组合[15]。研究结果表明，合理的RAG配置能够显著提升医学问答的准确性。在营养健康领域，Vassallo等利用基于RAG的LLM系统为肥胖和2型糖尿病患者提供个性化营养建议，通过整合循证医学数据库确保生成建议的科学性[5]。另一项研究则将RAG技术应用于孕期营养指导，采用LLaMA 2模型结合专家整理的营养数据集，为孕妇提供个性化的产前营养建议[16]。

向量数据库作为RAG系统的核心基础设施，近年来也取得了重要进展。PostgreSQL的pgvector扩展为关系型数据库增加了高效的向量存储和相似性搜索能力，支持余弦相似度、欧氏距离等多种相似度度量方法，并提供了HNSW和IVF等近似最近邻搜索索引[17]。pgvector的推出使得开发者能够在现有数据库基础设施上构建语义搜索功能，降低了RAG系统的部署门槛，已被广泛应用于医学知识检索、产品智能搜索等场景[18]。

#### 1.2.3 现有营养管理系统分析

当前市场上的营养管理类应用主要包括薄荷健康、Keep、FatSecret等商业化产品。这些应用普遍具备食物热量查询、饮食记录、体重管理等功能，部分产品还提供了社区交流和电商服务。然而，现有产品普遍存在以下不足：一是缺乏智能化的营养咨询能力，用户主要通过手动搜索获取信息，交互体验较差；二是个性化程度有限，推荐内容往往基于简单的规则匹配，难以根据用户的健康状况、饮食偏好和目标进行深度定制；三是知识更新滞后，营养知识库维护成本高，难以快速纳入最新的研究成果和膳食指南。

在学术研究方面，国内外学者对智能营养推荐系统进行了大量探索。传统的推荐方法主要基于协同过滤和内容过滤算法，利用用户的历史行为数据和食物的营养特征进行推荐[19][20]。近年来，随着深度学习技术的发展，研究者开始尝试将神经网络应用于营养推荐任务。例如，有研究利用知识图谱整合食物、营养成分和疾病之间的关系，实现可解释的健康饮食推荐[21]。然而，现有学术研究大多停留在算法模型层面，缺乏完整的系统实现和实际应用验证，与真实应用场景之间存在一定差距。

综上所述，现有营养管理工具在智能化水平、个性化能力和知识更新机制方面仍有较大提升空间。将大语言模型与RAG技术相结合，构建具备深度理解能力和权威知识支撑的智能营养顾问系统，是弥补上述不足的有效途径，也是本研究的核心动机所在。

### 1.3 研究内容与目标

#### 1.3.1 研究内容

本研究围绕基于大语言模型与RAG技术的智能营养顾问系统的设计与实现，主要开展以下研究工作：

**（1）基于LLM的营养咨询对话系统**
设计并实现基于大语言模型的营养咨询对话功能，支持用户通过自然语言与系统进行交互，获取营养知识、膳食建议和健康指导。研究提示词工程（Prompt Engineering）技术，设计适合营养咨询场景的提示词模板，引导模型生成专业、准确且易于理解的回答。

**（2）基于RAG的个性化食谱推荐算法**
研究并实现基于检索增强生成的个性化食谱推荐功能。通过构建营养知识向量库，结合用户的健康档案、饮食偏好和季节因素，利用语义检索获取最相关的食谱和食材知识，再通过大语言模型生成个性化的推荐结果和烹饪建议。

**（3）多模态食材识别与营养分析**
探索多模态视觉语言模型在食材识别场景中的应用。利用视觉语言模型（VLM）对上传的食材图片进行识别，提取食材种类和数量信息，结合营养成分数据库进行营养分析，为食谱推荐和膳食规划提供数据支持。

**（4）文档解析与向量知识库构建**
设计并实现文档解析与向量知识库构建系统，支持PDF、Word、TXT、Markdown等多种格式的文档上传与解析。研究文本分块策略（固定大小分块、句子分块、段落分块、递归分块），实现文档的自动分块、向量嵌入和入库管理，为RAG检索提供高质量的知识源。

**（5）前后端分离的全栈系统实现**
采用前后端分离架构，基于FastAPI框架搭建后端RESTful API服务，基于Vue 3和Vant UI框架构建移动端前端应用。实现用户认证、健康档案管理、对话历史记录、知识库管理等完整功能模块，并通过JWT认证、密码加密等机制保障系统安全。

#### 1.3.2 研究目标

本研究的主要目标包括：

**（1）构建完整可用的智能营养顾问平台**
开发一个功能完善、界面友好、运行稳定的智能营养顾问系统，涵盖营养咨询、食谱推荐、食材识别、健康档案管理和知识库管理等核心功能，满足用户在日常营养健康管理中的多样化需求。

**（2）实现高精度的语义知识检索与生成**
通过RAG技术将大语言模型与营养知识库深度融合，实现基于语义相似度的知识检索和基于上下文的智能回答生成，确保系统回答的专业性、准确性和可追溯性，有效降低模型幻觉现象。

**（3）提供友好的移动端交互体验**
针对移动端使用场景进行专门的UI/UX设计，确保系统在不同屏幕尺寸下的适配性和操作的便捷性。通过动画效果、进度提示、状态反馈等设计细节，提升用户的使用体验和满意度。

## 第 7 章 总结与展望

### 7.1 工作总结

- 完成了智能营养顾问系统的整体设计与实现
- 实现了基于 RAG 的知识检索与食谱推荐功能
- 构建了完整的文档解析与向量知识库管理系统
- 实现了前后端分离的全栈应用

### 7.2 创新点

- 将 RAG 技术应用于营养健康垂直领域
- 实现了关系数据库与向量数据库的双源融合
- 设计了完善的异步文档处理与进度持久化机制
- 提供了多模态食材识别与营养分析的集成方案

### 7.3 不足与改进方向

- RAG 检索精度仍有提升空间（可引入混合检索策略）
- 知识库规模有限（可扩展更多专业营养数据源）
- 移动端功能可进一步丰富（如饮食打卡、热量追踪）
- 可引入推荐算法实现更精准的个性化推荐

### 7.4 应用前景

- 个人日常营养健康管理工具
- 社区/医院营养咨询服务辅助平台
- 可扩展至其他健康垂直领域（如运动健身、慢性病管理）
